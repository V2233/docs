# 教程简介


:::tip

 Yunzai V4 是基于Icqq协议所构建的机器人开发框架，开发者可以通过开发插件来扩展机器人的功能。相较于从前的版本，V4的写法更加简洁，且拥有强大的TS&TSX环境，并基于模块化制作了图片热开发工具。总之，V4将从代码可维护性、开发效率、响应速度和程序稳定性等多方面进行升级。

:::
